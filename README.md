# VentaConecta

Plataforma que conecta comercios locales con vendedores independientes.

## Páginas

| Archivo | Descripción |
|--------|-------------|
| `index.html` | Landing page |
| `dashboard-comercio.html` | Dashboard para comercios |
| `dashboard-vendedor.html` | Dashboard para vendedores |
| `catalogo.html` | Catálogo de productos (vista vendedor) |
| `detalle-producto.html` | Detalle de producto + formulario de notificación |

## Stack
HTML · CSS · JavaScript vanilla · Fonts: Syne + DM Sans

## Deploy
Sube a Vercel directamente desde GitHub.
